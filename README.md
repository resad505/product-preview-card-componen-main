### What I learned

During this project, I improved my layout skills using CSS Flexbox and media queries for responsive design. I learned how to structure a card component to be fully responsive for both mobile and desktop screens. I also integrated custom fonts from Google Fonts and handled image switching for responsive modes.

### Continued development

In future projects, I would like to focus on further enhancing my CSS Grid skills and implementing more complex responsive layouts. I also plan to refine structural CSS naming conventions and semantic HTML tags.
